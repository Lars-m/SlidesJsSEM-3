Full Stack JavaScript - Content

  • Use JavaScript EVERYWHERE (client, server, App, Desktop - you name it)
  • UNDERSTAND and write your own Promise-based code.
  • Understand and use Babel & WebPack
  • Static Typing with TypeScript
  • Dig deeper into JavaScript and the newest features found in ES6,7...
  • Use Node.js + Express to write cool web/REST-based applications with JavaScript
  • Introduce GEO-JSON, and use cases involving geolocation
  • Use modern NoSQL databases like MongoDB to provide cool location based backends
  • Continue with Cross Platform Mobile Apps using React, REACT Native and JavaScript
  • Introduce GraphQL an modern Query Language for API's, by many seen as a "replacement" for REST
  • And if you like: WebSockets, Electron ....
  • Most of all: Have a lot of fun, and program a lot of Cool projects :-)

Form

The semester will be divided up into a number of periods, each focusing on a specific area of the syllabus. Each period, will start with a description of the learning goals and a a number of smaller exercises that has to be implemented before the end of the period.

The semester will also include an on-going mini-project to illustrate a Fullstack JavaScript Application involving:

  • Backend
    • JavaScript with Node and Express
    • Persistence with MongoDB
    • Geolocation with "plain" GeoJSON and MongoDB
    • Testing with Mocha and Chai
    • REST with Express
    • GraphQL
  • Frontend
    • React and React Native
    • GeoLocatioin via relevant React Native and Expo packages
    • GraphQL via Apollo
